# 🏦 Predictive Modeling and Risk Scoring for Bank Customer Churn

An advanced Machine Learning + Streamlit project designed to predict customer churn for a bank and generate actionable risk scores through an interactive dashboard.

This project helps banks identify customers likely to leave, understand key churn drivers, and take proactive retention actions.

---

## 🚀 Features

### 📊 Machine Learning Prediction
- Predict whether a customer will churn or stay
- Multiple model support (Logistic Regression, Random Forest, XGBoost)
- Probability-based churn scoring

### 🎯 Risk Scoring Engine
- Customer segmentation:
  - Low Risk
  - Medium Risk
  - High Risk
  - Critical Risk

### 📈 Interactive Dashboard
- KPI cards
- Customer churn trends
- Geography-wise churn analysis
- Age group segmentation
- Balance vs churn behavior
- Tenure insights

### 🧠 Explainable AI
- SHAP feature importance
- Customer-level prediction explanation
- Business-friendly insights

### 📂 Data Upload Support
- Upload custom CSV datasets
- Automatic preprocessing

### 🎨 Premium UI
- Modern Streamlit layout
- Sidebar controls
- Filters
- Responsive charts
